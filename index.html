Welcome to the Data Visualization Tools

On this site, you can perform the following actions:

  1. Upload Data: Click the "Upload File" button or manually enter data in the provided input field.
  2. Preview Data: After uploading data, view it in the "Preview" section.
  3. Build Charts: Select the chart type from the dropdown menu and click the "Draw Chart" button.
  4. Customize Charts: Use the settings panel to change colors, line styles, line thickness, and other parameters.
  5. Export Charts: Click the "Export as PNG", "Export as SVG", or "Export as PDF" buttons to save the chart.
  6. Print Charts: Click the "Print Chart" button to print the chart.
  7. Change Theme: Click the theme icon to switch between light and dark themes.
  8. Pan and Zoom: Use the mouse to pan and zoom the chart for a detailed view.
  9. Accessibility: Click the "Accessibility" button to adjust settings like cursor size or grayscale mode.
  10. View Tooltips: Hover over chart points to view tooltips.
  11. Display Settings: Use the settings panel to toggle the display of the legend, axes, and labels.

The site provides a convenient interface for working with charts and their customization, making it easy to analyze and visualize data.
